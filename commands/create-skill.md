# /create-skill

Crear un nuevo skill en el framework King.

## Instrucciones

1. Invocar el skill `create-skill`
2. El argumento es el nombre del nuevo skill a crear
3. Generar archivos con estructura GATE IN / MUST DO / CHECKPOINT
4. Crear matriz de participación de agentes
5. Registrar el skill en el framework
6. IMPORTANTE: Seguir convenciones existentes del King Framework
