# /frontend-design

Ejecutar workflow de diseño frontend moderno.

## Instrucciones

1. Invocar el skill `frontend-design`
2. El argumento describe qué interfaz diseñar
3. Seguir las fases: Inspiración → Wireframe → Design → Implementation → Review → Commit
4. Buscar inspiración en tendencias de diseño web 2026
5. Implementar con Tailwind CSS 4 y técnicas modernas (glassmorphism, gradientes, animaciones)
6. IMPORTANTE: Seguir las convenciones de estilos del proyecto (ver CLAUDE.md)
7. IMPORTANTE: Accesibilidad WCAG 2.1 AA mínimo

Si se proporciona `--workflow <nombre>`, asociar la ejecución a ese workflow. Si no se proporciona, se detecta automáticamente del branch actual.
