# /solid-check

Analiza el código del proyecto buscando violaciones SOLID (CASTLE A2 sub-score).

1. Usar el skill `skills/solid-check/SKILL.md`
2. Escanear el código fuente adaptándose al stack detectado
3. Escribir `.king/castle/solid-report.json`
4. El report es consumido automáticamente por `/castle-report` y `/qa` Phase 5
